## 描述

阅读 Claude 的 thinking 块以更好地理解其推理过程。

## 什么是 thinking 块

Thinking 块显示 Claude 的内部推理过程，让你看到：
- 它如何理解问题
- 考虑了哪些方案
- 为什么选择某个方案
- 识别到哪些风险

## 如何查看

```bash
# Ctrl+O (macOS: Option+T) 切换 thinking 显示
```

## 何时特别有用

- **复杂决策** - 了解推理过程
- **调试** - 看到分析思路
- **学习** - 学习如何思考
- **审查** - 验证 Claude 的逻辑

## 使用技巧

```
# 如果结果不对，检查 thinking
"你的 thinking 中，为什么假设了 X？"

# 利用 thinking 迭代
"在 thinking 中重新考虑方案 Y"
```
