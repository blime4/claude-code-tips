## 描述

使用 `/help` 命令查看所有可用的斜杠命令。

## 使用方法

```bash
/help
```

## 常用命令速查

| 命令 | 功能 |
|------|------|
| `/init` | 初始化项目配置 |
| `/clear` | 清空对话 |
| `/context` | 显示 token 使用 |
| `/model` | 切换模型 |
| `/resume` | 恢复会话 |
| `/mcp` | MCP 状态 |
| `/help` | 显示帮助 |

## 学习建议

- 新手：先记住 `/help` 和 `/clear`
- 日常：加上 `/context` 和 `/model`
- 高级：掌握所有命令
