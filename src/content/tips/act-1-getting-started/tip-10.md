## 描述

可以直接将截图拖拽到 Claude Code 中，Claude 会"看到"图片内容。

## 使用方法

1. 截取屏幕截图
2. 直接拖拽到 Claude Code 窗口
3. Claude 会分析图片内容

## 适用场景

- 错误信息截图
- UI 设计稿
- 代码截图
- 架构图/流程图
