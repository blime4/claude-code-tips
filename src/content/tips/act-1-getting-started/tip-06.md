## 描述

使用 `Shift+Tab` 在 Ask 模式和 Command 模式之间切换。

## 模式说明

- **Ask 模式**: 默认模式，用于对话和提问
- **Command 模式**: 直接执行命令，无需确认

## 使用场景

```bash
# Ask 模式（默认）
> 帮我创建一个新的组件
# Claude 会询问确认

# Command 模式（Shift+Tab）
> 创建用户卡片组件
# 直接执行，不询问
```
