## 描述

使用 `/clear` 命令重置对话上下文，开始新对话。

## 使用方法

```bash
/clear
```

## 使用场景

- 切换到完全不同的任务
- 上下文过于混乱
- 想要重新开始
- 测试不同的方法

## 注意

- `/clear` 会清除所有对话历史
- 但不会影响已修改的文件
- 不会修改 `CLAUDE.md` 配置
