快速入门

5 分钟上手 Claude Code

1. 安装

npm install -g @anthropics/claude-code

2. 首次运行

# 进入你的项目目录
cd my-project

# 启动 Claude Code
claude

3. 初始化配置

# 在 Claude Code 中运行
/init

这将扫描项目并创建 `CLAUDE.md` 配置文件。

4. 基本命令

命令 功能
/help显示所有命令
/clear清空对话
/context查看 token 使用
/model切换模型

5. 开始使用

现在你可以：

  • 用自然语言描述你想做什么
  • 拖拽截图进行分析
  • 让 Claude 阅读、修改、创建文件
