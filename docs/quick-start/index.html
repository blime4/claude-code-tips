快速入门

5 分钟上手 Claude Code

1. 安装

方法一：npm 安装（推荐）

中国大陆��户：使用淘宝镜像加速

# 配置 npm 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

其他地区：

npm install -g @anthropic-ai/claude-code
权限问题解决方案

如果遇到 EACCES 权限错误，可以安装到用户目录：

mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
npm install -g @anthropic-ai/claude-code

# 添加到 PATH（添加到 ~/.bashrc 或 ~/.zshrc）
export PATH=~/.npm-global/bin:$PATH

方法二：原生安装脚本

# macOS、Linux、WSL
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex

方法三：Homebrew（macOS）

brew install --cask claude-code

2. 首次运行

# 进入你的项目目录
cd my-project

# 启动 Claude Code
claude

3. 初始化配置

# 在 Claude Code 中运行
/init

这将扫描项目并创建 `CLAUDE.md` 配置文件。

4. 基本命令

命令 功能
/help显示所有命令
/clear清空对话
/context查看 token 使用
/model切换模型

5. 开始使用

现在你可以：

  • 用自然语言描述你想做什么
  • 拖拽截图进行分析
  • 让 Claude 阅读、修改、创建文件
