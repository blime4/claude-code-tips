Tip #15 intermediate Act 1

/model switches models

Use /model to switch between Claude models

⏱ 11:42 📁 workflow
标签： commandsmodelsoptimization

描述

使用 /model 命令在不同 Claude 模型之间切换。

使用方法

/model sonnet    # 默认，最佳编程模型
/model opus      # 最强推理能力
/model haiku     # 最快，轻量任务

模型选择指南

模型用途特点
Haiku轻量任务快，便宜，90% 能力
Sonnet日常开发平衡，推荐默认
Opus复杂任务最强推理，较慢

使用场景

# 简单查询
/model haiku
> 这个函数是做什么的？

# 日常开发
/model sonnet  # 默认
> 帮我实现用户认证

# 复杂架构决策
/model opus
> 设计一个支持千万用户的系统架构
