Enable notifications

Enable notifications for long-running tasks

⏱ 34:10 📁 advanced
标签： notificationsproductivitysettings

描述

为长时间运行的任务启用通知。

配置方法

在 ~/.claude/settings.json：

{
  "notifications": {
    "enabled": true,
    "onComplete": true,
    "onError": true,
    "sound": true
  }
}

使用场景

  • 长时间构建
  • 大型重构
  • 测试运行
  • Agent 执行

工作流程

# 1. 启动任务
"运行完整测试套件"

# 2. 切换到其他工作
# 3. 收到通知后返回

通知类型

  • ✅ 任务完成
  • ❌ 任务失败
  • 🔔 需要 user input
  • 💡 有建议
