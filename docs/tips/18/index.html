/help shows all commands

Use /help to see all available commands

⏱ 13:21 📁 setup
标签： commandshelpreference

描述

使用 /help 命令查看所有可用的斜杠命令。

使用方法

/help

常用命令速查

命令功能
/init初始化项目配置
/clear清空对话
/context显示 token 使用
/model切换模型
/resume恢复会话
/mcpMCP 状态
/help显示帮助

学习建议

  • 新手：先记住 /help 和 /clear
  • 日常：加上 /context 和 /model
  • 高级：掌握所有命令
