/resume recovers sessions

Use /resume to recover previous sessions

⏱ 12:23 📁 workflow
标签： commandssessionsrecovery

描述

使用 /resume 命令恢复之前的会话状态。

使用方法

/resume
# 或指定会话ID
/resume <session-id>

使用场景

  • 意外关闭了 Claude Code
  • 系统崩溃后恢复
  • 在不同设备间继续
  • 查看历史会话

工作原理

会话保存在: ~/.claude/sessions/

恢复时:
- 恢复对话历史
- 恢复上下文
- 恢复项目状态

注意

  • 会话有保留期限
  • 文件修改已保存，不会丢失
  • 只恢复对话上下文
