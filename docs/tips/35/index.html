Tip #35 beginner Act 3

Commands = quick shorthand

Commands are quick aliases for common actions

⏱ 26:50 📁 power-user
标签： commandsshortcutsproductivity

描述

Commands 是常用操作的快捷别名。

常用 Commands

/commit      # 提交代码
/pr          # 创建 PR
/review      # 代码审查
/test        # 运行测试
/deploy      # 部署

自定义 Command

// ~/.claude/commands/deploy-staging.json
{
  "command": "deploy-staging",
  "description": "部署到 staging 环境",
  "action": "bash",
  "template": "npm run build && npm run deploy:staging"
}

Command vs Skill

特性CommandSkill
复杂度简单复杂
步骤单步多步
交互少多
用途快捷方式完整流程
