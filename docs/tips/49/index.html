Block dangerous commands

Use hooks to block dangerous commands

⏱ 40:10 📁 advanced
标签： hookssecuritysafety

描述

使用 hooks 阻止危险命令执行。

危险命令模式

rm -rf
git clean -fdx
:(){ :|:& };:    # fork bomb
dd if=/dev/zero
> /dev/sda       # 覆盖磁盘

Hook 配置

{
  "hooks": {
    "PreToolUse": {
      "Bash": [
        "if echo $COMMAND | grep -q 'rm -rf'; then",
        "  echo '❌ DANGEROUS: rm -rf blocked'",
        "  exit 1",
        "fi"
      ]
    }
  }
}

更智能的检查

# 检查脚本
if [[ $FILE == *.sh ]]; then
  if grep -q 'rm -rf' $FILE; then
    echo "⚠️ Script contains rm -rf"
    read -p "Continue? (y/N) " -n 1 -r
    if [[ ! $REPLY =~ ^[Yy]$ ]]; then
      exit 1
    fi
  fi
fi
