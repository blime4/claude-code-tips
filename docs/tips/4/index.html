Keep CLAUDE.md concise

Keep CLAUDE.md files short and focused

⏱ 3:27 📁 setup
标签： claudemdconfigdocumentation

描述

保持 CLAUDE.md 文件简短专注，只包含项目特定的关键信息。

原则

  • YAGNI: 只写当前需要的内容
  • 避免重复: 不要重复全局规则中已有的内容
  • 聚焦差异: 只写这个项目与其他项目不同的地方

好的 CLAUDE.md

# My Project

## 项目特点
- 使用 FastAPI 而非 Django
- 遵循 PEP 8 规范
- 测试覆盖率要求 90%+
